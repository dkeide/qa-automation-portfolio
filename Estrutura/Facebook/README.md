Automação de Cadastro - Facebook (Robot Framework)
📌 Descrição

Projeto de automação de testes simulando o fluxo de cadastro de um novo usuário.

🚀 Tecnologias utilizadas
Robot Framework
SeleniumLibrary
Python
🧪 O que foi testado
Preenchimento de formulário
Validação de campos obrigatórios
Fluxo completo de cadastro
📂 Estrutura do projeto
resources → arquivos reutilizáveis
suites → cenários de teste
results → evidências dos testes
▶️ Como executar
Instalar dependências:
pip install -r requirements.txt
Rodar os testes:
robot suites/
📊 Resultados

Os relatórios são gerados automaticamente na pasta "results".
